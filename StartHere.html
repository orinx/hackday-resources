Web App Templates

  • mortar app stub: for any app; a blank template that comes only with the basic things required to get started | Download
  • mortar game stub: for building 2D games with canvas; a minimal web game template that handles input and a render loop | Download
  • mortar list-detail layout: for list-based apps; comes with a layouts library and sets up a list-detail layout for you | Download
  • mortar tab view: similar to mortar-list-detail but the initial layout has tabs at the button which let you switch between screens. | Download
  • sutthisan: for apps that need a simple server-side component | Download
